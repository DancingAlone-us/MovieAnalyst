Movie Industry Insights & Dashboard

The objective of this project was to learn and test my understanding on Pandas, Numpy, Matplotlib and PowerBI. 

I used a csv file which I got from kaggle, the dataset was thankfully of netflix. The link to the dataset is : https://www.kaggle.com/datasets/sanyamdhadiwal2302/netflix-data

I used Pandas and Numpy for basic operations and used some Matplotlib for the visualization of datas.

From my operations, I found out that animation / animated films are the most popular genre in the netflix website, followed by Sci-Fi, Action, Thriller and Crime. Similarly, apart from english, the most famous language for any movie is japanese.
